This module provides code for benchmarking the performance of optimization methods and for simulating datasets.

**Dependencies**

The following dependencies are required:

- numpy == 1.23.4
- matplotlib == 3.6.2
- scikit-learn == 1.1.3
- torch == 1.13.0
- jupyter == 1.0.0
- tqdm == 4.64.1
- pandas == 1.5.2


**Installation**
```
$ git clone https://github.com/ryanhammonds/dsc210
$ pip install -e .
```
